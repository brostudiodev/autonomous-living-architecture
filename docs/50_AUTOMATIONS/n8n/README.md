# n8n Automations

## Storage convention
- `workflows/WF###__name.json` = exported workflow
- `workflows/WF###__name.md` = human-readable spec

## Minimal workflow doc template
- Purpose
- Trigger(s)
- Inputs / outputs
- Credentials used (names only, no values)
- Dependencies (services/APIs)
- Failure modes + alerts
- Test procedure
- Rollback
