## 🔍 Exploration Paths

### If you have a Goal
Go to: `docs/10_Goals/`
- Each goal (e.g., `G01_Target-Body-Fat`) has a `README.md` explaining "What you achieve".
- Follow links within those READMEs to see the enabling systems.

### If you want to see the "Engine" (Systems)
Go to: `docs/20_Systems/`
- Technical specifications for the building blocks like the `S04_Digital-Twin` or `S03_Data-Layer`.
- See the **[Tooling Path Comparison](./Tooling-Path-Comparison.md)** to understand my choice of tools.

### If you need instructions (SOPs)
Go to: `docs/30_Sops/`
- Step-by-step guides for recurring tasks.

### If something broke
Go to: `docs/40_Runbooks/`
- Incident response and troubleshooting guides.

### If you’re looking for automation logic
Go to: `docs/50_Automations/`
- Documentation for Python scripts and n8n workflows.
