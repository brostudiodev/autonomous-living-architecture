# External LLM Prompts

Use these when the LLM does **not** have access to this repository.

## How to use (recommended)
1. Copy/paste the generator prompt: `External-LLM-Generator-Prompt.md`
2. Copy/paste one goal request template from `Goal-Requests/` and fill the TBD fields.
