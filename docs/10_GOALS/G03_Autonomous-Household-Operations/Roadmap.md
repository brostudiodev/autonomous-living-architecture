# Roadmap (2026)

- [x] Pantry Management System v1.0 Deployed
- [ ] Integrate pantry consumption data with grocery automation (WF101)

- [ ] Begin feeding household operational data into G12 (Meta-System) for cross-system optimization
- [ ] Expand pantry system to include expiration date tracking and alerts

- [ ] Develop automated task assignment for household chores
- [ ] Implement energy consumption monitoring for key appliances
- [ ] Connect utility bill data from G05 for cost optimization insights
- [ ] Automate household maintenance reminders and scheduling

## Q3 (Jul–Sep)
- [ ] Integrate smart garden/plant care systems
- [ ] Implement predictive maintenance for household infrastructure (e.g., HVAC, water heater)

## Q4 (Oct–Dec)
- [ ] Full integration with G08 (Predictive Smart Home Orchestration) for holistic management
- [ ] Implement AI-driven optimization for daily household routines
- [ ] Create a comprehensive household operational dashboard
- [ ] Finalize data flow and insights contribution to G12 (Meta-System)
- [ ] Document lessons learned and strategy for 2027

## Dependencies
- **Systems:** S03 (Data Layer for inventory, sensor data), S05 (Finance for budget integration), S07 (Smart Home for device control), S08 (Automation Orchestrator for workflows)
- **External:** Google Sheets API, n8n, various smart device APIs
- **Other goals:** G05 (Autonomous Finance) for budget, G08 (Smart Home Orchestration) for device integration, G12 (Meta-System) for overall optimization.
