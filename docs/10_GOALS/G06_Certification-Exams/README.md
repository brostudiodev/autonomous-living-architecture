# G06: Certification Exams

## Purpose
Achieve targeted professional certifications that validate skills, enhance career credibility, and provide structured learning paths for acquiring expertise in cloud platforms, cybersecurity, AI/ML, and other relevant domains.

## Scope
### In Scope
- Cloud platform certifications (AWS, GCP, Azure)
- Cybersecurity certifications
- AI/ML certifications
- Structured study plans
- Progress tracking integration

### Out of Scope
- Degree programs
- Non-technical certifications
- Paid exam prep courses (use free resources)

## Intent
Achieve targeted professional certifications that validate skills and enhance career credibility through structured learning and measurable milestones.

## Definition of Done (2026)
- [ ] Target certifications identified
- [ ] Study materials researched
- [ ] Structured study plan with milestones
- [ ] First certification completed (if Q1)
- [ ] Learning resources documented

## Inputs
- Career goals and skill gaps
- Available study resources (free)
- Exam provider requirements
- Time availability

## Outputs
- Completed certifications
- Documented study strategies
- Updated skill profile in G10
- Case studies for portfolio

## Dependencies
### Systems
- G10 Intelligent Productivity (study scheduling)
- G09 Career Intelligence (skill tracking)

### External
- Exam provider platforms
- Online learning resources
- Practice exam sites

## Procedure
1. **Quarterly:** Identify target certifications based on career goals
2. **Before exam:** Research study materials, create study plan
3. **Weekly:** Study according to plan, track progress
4. **After exam:** Document results, update G10

## Failure Modes
| Scenario | Detection | Response |
|----------|-----------|----------|
| Exam fails | Score below passing | Review weak areas, reschedule |
| Study stalls | No progress in 2+ weeks | Adjust plan, reduce load |
| Resources inadequate | Can't find good materials | Explore alternative sources |

## Security Notes
- Exam credentials stored securely
- No sensitive personal data shared with exam providers
