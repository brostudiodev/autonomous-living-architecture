# SOPs (Standard Operating Procedures)

## Purpose
Document repeatable life operations so they can be:
- executed manually
- automated safely
- audited and improved

## Format standard
Each SOP should include:
- Trigger / frequency
- Inputs
- Steps (checklist)
- Outputs
- Failure modes and escalation
