# SOP: G01 Monthly Review

## Purpose
Analyze monthly progress toward target body fat goals.

## Procedure
1. Run `G01_monthly_reporter.py`.
2. Review the generated report in `artifacts/reports/`.
3. Compare body fat delta with monthly target.
4. Adjust nutrition/training parameters if needed.
