# SOP: Daily Review

## Purpose
Daily review of tasks, goals, and system status to maintain productivity and catch issues early.

## Procedure
1. Open today's Daily Note in Obsidian
2. Review autonomous suggestions in `🤖 Autonomous Task Suggestions`
3. Check for budget/health alerts
4. Complete and check off tasks
5. End-of-day: Run evening automation (`autonomous_evening_manager.py` at 18:00)
