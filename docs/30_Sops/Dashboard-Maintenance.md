# SOP: Dashboard Maintenance

## Purpose
Ensure Grafana dashboards remain accurate and high-performance.

## Procedure
1. Check for broken panels or "No Data" alerts.
2. Verify dashboard variables pull correct values from DB.
3. Export JSON backup of updated dashboards.
4. Clean up unused or redundant panels.
