# SOP: AI Agent Troubleshooting

## Purpose
Resolve issues with Gemini-powered personal agents.

## Procedure
1. Check n8n execution log for failed nodes.
2. Verify API quota for Google Gemini.
3. Inspect input prompt formatting in the router.
4. Test with a simplified prompt to isolate logic vs. API issues.
