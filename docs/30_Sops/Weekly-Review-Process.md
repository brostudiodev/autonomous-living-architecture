# SOP: Weekly Review Process

## Purpose
Guide the transition from daily execution to weekly strategic reflection.

## Procedure
1. Run `autonomous_weekly_manager.py` every Sunday morning.
2. Review the generated Obsidian Weekly Note.
3. Update "Starved Goals" status.
4. Set top 3 priorities for the coming week.
