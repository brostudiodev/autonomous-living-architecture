# SOP: Weekly Review Process

## Purpose
Guide the transition from daily execution to weekly strategic reflection.

## Automated Flow (Sundays)

| Time | Automation | Output |
|------|------------|--------|
| 06:00 | `autonomous_weekly_manager.py` | Obsidian Weekly Note |
| 08:00 | `G11_ceo_weekly_briefing.py` | Telegram Executive Summary |
| 10:00 | `G11_task_archiver.py` | Stale task cleanup |
| 21:00 | `G10_weekly_rollup.py` | Weekly trend analysis |

## Manual Review (Optional)

1. Review the **Telegram CEO Briefing** received at 8 AM
2. Open the **Obsidian Weekly Note** for detailed analysis
3. Update "Starved Goals" status if needed
4. Set top 3 priorities for the coming week

## Enhanced Briefing Content

The CEO Weekly Briefing includes:
- Finance: Savings rate, MTD net, budget alerts
- Health: Readiness, sleep, weight, training
- Productivity: Goals touched, time saved
- Tasks: Overdue status, pending approvals
- Next week priorities based on patterns
