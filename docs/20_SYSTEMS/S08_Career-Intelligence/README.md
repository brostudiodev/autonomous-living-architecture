# S08: Career Intelligence

## Purpose
Build automated career intelligence system for continuous market opportunity tracking by Dec 31, 2026. Maintain professional profile positioning for enterprise AI/automation roles, automatically identify skill gaps and suggest learning priorities, and manage strategic networking and application processes.

## Scope
- Included: Automated job board scraping, AI-powered requirement analysis, Skill gap identification, LinkedIn profile automation, Application follow-up sequences, Industry trend monitoring
- Excluded: Resume/CV writing services, Interview coaching, Direct recruiter outreach, Salary negotiation advice

## Interfaces
- Inputs: Job postings from multiple platforms, Company career pages, Professional networking data, Industry reports, Personal skill assessments, Learning history
- Outputs: High-fit role matches, Skill gap analyses, Learning recommendations, LinkedIn content suggestions, Application follow-ups, Market trend insights
- APIs/events: Job board APIs (LinkedIn, Indeed), Company career page scrapers, Claude API (requirement analysis), LinkedIn API (profile management), Email automation APIs, Calendar integration

## Dependencies
- Services: Job board scraping services, Claude API for analysis, LinkedIn API for profile management, Email automation platforms, Calendar integration service
- Hardware: API access endpoints, Data storage for tracking applications, AI processing capability (local or cloud)
- Credentials (names only): linkedin_api_key, job_board_api_keys, claude_api_key, email_automation_credentials, calendar_api_credentials

## Observability
- Logs: Job scraping execution logs, Analysis job processing logs, LinkedIn post success rates, Application tracking updates, API rate limiting events
- Metrics: Weekly role-fit score, High-fit roles found per week, Application response rates, LinkedIn engagement metrics, Learning completion rates, Skill gap resolution
- Alerts: New high-fit role matches, Application deadlines approaching, LinkedIn engagement anomalies, API quota limits, System scraping failures

## Runbooks / SOPs
- Related SOPs: Job board API setup, LinkedIn profile optimization, Application tracking procedures, Learning plan creation, Professional content scheduling
- Related runbooks: Application response management, Interview scheduling automation, Professional network engagement, Market analysis procedures
